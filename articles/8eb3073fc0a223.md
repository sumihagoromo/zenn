## 1. 🎀 すみ流！おすすめのディレクトリ構成

Vault（保管庫）をそのままGit管理するんじゃなくて、リポジトリの中に**「目的別のVault」**をいくつか入れるのがポイントだよっ✨

```text
MyObsidianRepo/         <-- ここをGit管理（GitHubリポジトリ）
├── .gitignore          <-- 共通の「これはいらないよ！」設定
├── MainVault/          <-- いつものお勉強メモ用📝
├── PromptLibrary/      <-- 大量に保存したプロンプトくんたち🤖
├── ArchiveVault/       <-- お蔵入りした古いエモい日記とか📖
└── obsidian_git.sh     <-- 同期用の魔法の杖（スクリプト）

```

### なんで分けるの？

* **お片付けがラクちん！**：使わなくなったメモを `ArchiveVault/` にポイッて移動するだけで、履歴を残したままスッキリ整理できちゃうの✨
* **AIくんが賢くなる！**：「プロンプト集だけ整理して！」ってAIくんにお願いしやすくなるんだよ〜💖

## 2. 🪄 魔法のスクリプト（obsidian_git.sh）を作ろう

iCloudを使ってる人はパスにスペースがあるから注意してね！でも大丈夫、すみがお手本を書いてあげたからっ✨

**`obsidian_git.sh`**（リポジトリの直下に置いてね！）

```bash
#!/bin/bash

# ⚠️自分のパスに書き換えてね！
REPO_PATH="/Users/おなまえ/Library/Mobile Documents/iCloud~md~obsidian/Documents"

cd "$REPO_PATH"

# 変更があったら、AIくん宛にコミット＆プッシュ！🚀
if [[ -n $(git status -s) ]]; then
    git add .
    git commit -m "Auto-sync: $(date '+%Y-%m-%d %H:%M:%S') ✨"
    git push origin main
fi

```

※作ったらターミナルで `chmod +x obsidian_git.sh` って唱えるのを忘れないでねっ！🪄

## 3. ⏰ launchdちゃんで「2時間おき」に自動化！

Macの `launchd` ちゃんは、PCを閉じてスリープしてても、**「起きた瞬間」に溜まってた分をすぐ実行してくれる**とってもいい子なのっ！(∩ˊ꒳​ˋ∩)･*

**`~/Library/LaunchAgents/com.obsidian.sync.plist`** を作ってこれを入れてね！

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
    <key>Label</key>
    <string>com.obsidian.sync</string>
    <key>ProgramArguments</key>
    <array>
        <string>/bin/bash</string>
        <string>/Users/rysh/Library/Mobile Documents/iCloud~md~obsidian/Documents/obsidian_git.sh</string>
    </array>
    <key>StartInterval</key>
    <integer>7200</integer> <key>RunAtLoad</key>
    <true/>
    <key>StandardOutPath</key>
    <string>/tmp/obsidian_sync.log</string>
    <key>StandardErrorPath</key>
    <string>/tmp/obsidian_sync.err</string>
</dict>
</plist>

```

書き終わったら、ターミナルで `launchctl load ~/Library/LaunchAgents/com.obsidian.sync.plist` って叩けば、自動化スタートだよっ！🎉
